# Welcome HΩme Healing Arts — Website

## Edit the Site
- Update text directly in **index.html** (Intro, Services, Events, Testimonials, Book).
- Replace the video embed in `index.html` (search for `YOUR_VIDEO_ID`) with your real YouTube/Vimeo embed.
- Calendly is set to **https://calendly.com/tylernegrete**.
- Replace `CITY_NAME` in the Book section when ready.
- Ensure your logo file is named **WHlogo.png** and uploaded to the repo root.

## Host on GitHub Pages (Web)
1. Create a public repo (e.g., `welcome-home-healing-arts`).
2. Add files: **index.html**, **style.css**, **script.js**, **WHlogo.png**, **README.md**.
3. **Settings → Pages** → Source: **Deploy from a branch** → Branch: **main** / **/** (root).
4. Open the URL GitHub shows (give it ~1 minute to publish).
