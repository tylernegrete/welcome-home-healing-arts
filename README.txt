# Welcome HΩme Healing Arts Website

This is a fully coded one-page website designed for **Tyler Negrete**'s healing practice.

### Hosting Instructions
1. Download the ZIP file and extract it.
2. Go to your GitHub repository.
3. Upload the files (index.html, banner.png, temple-bg.jpg).
4. Enable GitHub Pages in your repo settings under **Pages → Branch → main/root**.
5. Visit your live site!

### Editable Sections
You can update text directly in the index.html file under the <section> tags.
